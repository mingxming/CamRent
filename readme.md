# 相机租赁日期记录系统

## 项目结构

- `server.js`：主服务器文件
- `dist`：前端构建输出目录
- `package.json`：项目配置文件
- `package-lock.json`：依赖版本锁定文件
- `README.md`：项目说明文件

## 项目依赖

- `express`：Web框架
- `react`：React框架
- `react-dom`：React DOM库
- `react-router-dom`：React路由库
- `axios`：HTTP客户端
- `moment`：时间处理库
